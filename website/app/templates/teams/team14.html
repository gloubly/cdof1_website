Infrared sensor

After doing some researches to learn more about infrared our subject, we where give an infrared sensor along with an Arduino.
We developed a code that allows us to visualize in real-time a heatmap from data data from the sensor. It is able to detect a range of temperatures between 20°C and 40°C.
In a second time, we trained an object detection model using a dataset of landmines infrared images found online. We annotated the images, used data augmentation to increase the dataset's size and trained the model which gave us very promising results.
